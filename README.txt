How to Run Schedule Smart
=========================

1. Run the schedule.py program with IDLE 3.7
2. Click the "Create New Sheet" button
3. Select the fields you want in the schedule
4. Click "Select Fields" button
5. Click "Create Sheet" button
6. Find the new schedule.xlsx spreadsheet in the schedules folder with the program and open it
7. Note that the "Autofill", "Edit", and "Update" buttons have no function yet. They are just placeholders for new things in the future.
